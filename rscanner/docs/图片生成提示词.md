## 使用方式

将以上 6 个提示词分别发送给 GPT（支持图片生成的版本），生成后保存到 `rscanner/docs/images/` 目录下：

```
rscanner/docs/images/
├── 01-界面布局.png
├── 02-扫描进行中.png
├── 03-发现违规.png
├── 04-无违规.png
├── 05-右键菜单.png
└── 06-上报流程.png
```
