# 违规软件检测工具 V6 — 使用说明

> 基于 Rust + Tauri 构建的企业级终端合规检测工具

---

## 一、工具简介

违规软件检测工具 V6 是一款轻量级桌面应用，用于检测终端设备上是否安装了违规软件（如钉钉等）。检测完成后自动将结果上报至企业微信和智能表，实现合规闭环。

**技术栈：** Rust（后端引擎）+ Tauri 2（桌面框架）+ React + TypeScript + Tailwind CSS

---

## 二、界面布局总览

界面采用深色 Material Design 3 风格，分为以下区域：

| 区域 | 功能 |
|------|------|
| **顶部栏** | 姓名输入 + 操作按钮组 |
| **左侧边栏** | 统计面板 + 实时日志 |
| **中央区域** | 三阶段进度条 + 扫描结果树 |
| **底部状态栏** | 当前阶段 + 选中数量 |

## 三、操作流程

### 步骤 1：输入姓名

在顶部右侧的输入框中输入检测人员姓名（必填）。姓名将作为上报记录的唯一标识。

```
提示：未输入姓名时，开始按钮不可用
```

### 步骤 2：点击「开始扫描」

点击 ▶ 按钮启动扫描。工具自动执行三个阶段：

| 阶段 | 说明 | 耗时参考 |
|------|------|----------|
| 🔍 **注册表扫描** | 检查 Windows 注册表中的安装记录 | < 1 秒 |
| 📁 **固定路径扫描** | 检查 Program Files、AppData 等常见安装目录 | 1-3 秒 |
| 💽 **全盘扫描** | 递归扫描所有磁盘分区（最彻底） | 取决于磁盘大小 |

### 步骤 3：查看结果

扫描过程中，右侧日志面板实时显示进度。完成后：

- **有违规** → 结果树显示检测到的文件，按目录分组
- **无违规** → 显示绿色状态，自动上报"无违规"记录

### 步骤 4：自动上报

当检测到违规项时，工具自动执行以下操作：

1. 通过 IP 获取当前城市位置
2. 向企业微信 Webhook 发送 Markdown 格式报告
3. 向智能表追加一条记录（含人员、时间、数量、位置）

```
上报内容包括：人员姓名、主机名、IP 所在城市、检测时间、违规数量、文件类型统计、文件列表（前 20 条）
```

## 四、高级操作

### 4.1 右键菜单

在结果列表中右键点击任意文件，可执行：

- **在资源管理器中打开** — 定位到文件所在目录
- **复制路径** — 将完整路径复制到剪贴板

### 4.2 批量操作

- **Ctrl+A** — 全选所有检测结果
- **Delete** — 将选中文件移至回收站（可恢复）
- **取消全选** — 清除所有选中状态

### 4.3 取消扫描

扫描过程中点击 ■ 按钮可随时中断，已扫描的结果仍会保留并上报。

### 4.4 上报状态反馈

上报完成后，侧边栏日志会显示详细结果：

```
[14:30:01] 扫描完成，发现 12 项
[14:30:01] 正在获取位置信息...
[14:30:02] 位置: 中国 江苏省 南京市
[14:30:02] 正在自动上报...
[14:30:03] 上报完成
```

若智能表上报失败（如 Webhook 过期），企业微信通知仍会成功，日志会显示：

```
[14:30:03] 上报失败: 企业微信通知成功，智能表上报失败: 840001 invalid webhook
```

## 五、检测规则说明

### 文件名关键词匹配

工具通过以下关键词识别违规软件：

| 匹配词 | 类型 |
|--------|------|
| `钉钉` / `dingding` / `dingtalk` | 中英文关键词 |
| `.exe` / `.msi` / `.apk` | 安装包 |
| `.pdf` / `.xlsx` / `.docx` | 文档 |
| `.zip` / `.rar` / `.7z` | 压缩包 |

### 误报过滤

以下路径自动排除（避免误报）：

- `kingsoft` / `wps` 目录下的同名图标文件
- `.svg` / `.ico` / `.png` / `.jpg` 等图片格式
- `.lnk` 快捷方式文件
- `.log` / `.txt` 日志和临时文件

## 六、常见问题

### Q：扫描很慢怎么办？
A：全盘扫描耗时取决于磁盘文件数量。首次扫描较慢，后续扫描有系统缓存加速。可先观察进度条中的速度和已扫描文件数。

### Q：IP 位置显示"未知"？
A：需要网络连接。若公司网络限制了外部 API 访问，位置功能会超时返回"未知"，不影响其他功能。

### Q：企业微信没收到通知？
A：检查 Webhook 地址是否过期。可在企业微信群设置中重新生成 Webhook。

### Q：如何恢复误删的文件？
A：工具使用回收站删除，可在 Windows 回收站中右键还原。

## 七、部署说明

### 环境要求
- Windows 10/11（64 位）
- 无需安装运行时依赖，单 exe 即可运行

### 编译构建
```bash
cd rscanner
npm install
cargo tauri build
```

产物路径：`rscanner/target/release/rscanner.exe`

## 八、技术架构

```
rscanner/
├── src-tauri/           # Rust 后端
│   ├── src/lib.rs       # Tauri 命令入口（8 个 IPC 命令）
│   ├── src/main.rs      # 程序入口
│   └── Cargo.toml       # Rust 依赖
├── crates/
│   ├── scanner-core/    # 检测引擎（注册表 + 路径 + 全盘扫描）
│   └── notifier/        # 上报模块（企业微信 + 智能表 + IP 定位）
├── src/                 # React 前端
│   ├── App.tsx          # 主界面逻辑
│   └── components/      # 8 个 UI 组件
└── dist/                # 前端构建产物
```

### Tauri IPC 命令列表

| 命令 | 功能 |
|------|------|
| `start_scan` | 启动后台扫描 |
| `cancel_scan` | 取消扫描 |
| `get_drives` | 获取可用磁盘分区 |
| `delete_files` | 删除文件至回收站 |
| `clean_registry` | 清理注册表 |
| `open_in_explorer` | 在资源管理器中打开 |
| `get_city_from_ip` | 通过 IP 获取城市 |
| `send_report` | 上报结果（微信 + 智能表） |
