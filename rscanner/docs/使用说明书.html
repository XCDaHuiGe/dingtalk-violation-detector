一、工具简介

违规软件检测工具 V6 是一款轻量级桌面应用，用于检测终端设备上是否安装了违规软件（如钉钉等）。检测完成后自动将结果上报至企业微信和智能表，实现合规闭环。

技术栈： Rust（后端引擎）+ Tauri 2（桌面框架）+ React + TypeScript + Tailwind CSS
运行要求： Windows 10/11 64 位，无需安装任何运行时
特性说明
⚡ 单 exe 分发无依赖，双击即用
🔍 三重检测注册表 + 固定路径 + 全盘扫描
🤖 自动上报扫描完成自动推送企业微信 + 智能表
🌐 IP 定位自动获取城市信息附加到报告
♻️ 安全删除文件移至回收站，可恢复

三、操作流程

1

输入姓名

在顶部右侧输入框中输入检测人员姓名。姓名将作为上报记录的唯一标识。未输入姓名时，开始按钮不可用。

2

点击「开始扫描」

点击 ▶ 按钮启动扫描。工具自动执行三个阶段：

阶段检测内容耗时参考
🔍 注册表扫描检查 Windows 注册表中的安装记录< 1 秒
📁 固定路径扫描检查 Program Files、AppData 等目录1-3 秒
💽 全盘扫描递归扫描所有磁盘分区取决于磁盘大小
3

查看结果

扫描完成后，结果按目录分组显示在中央区域。每个文件可看到：文件名（可点击选中）和文件类型标签（安装程序/文档/压缩包）。

4

自动上报（有违规时触发）

当检测到违规项时，工具自动完成上报，无需手动操作：

① 通过 IP 获取当前城市位置 → ② 向企业微信 Webhook 发送报告 → ③ 向智能表追加记录

七、常见问题

❓ 扫描很慢怎么办？

全盘扫描耗时取决于磁盘文件数量。首次扫描较慢，后续有系统缓存加速。可先观察进度条中的速度和已扫描文件数判断是否正常。

❓ IP 位置显示"未知"或"未知（超时）"？

需要网络连接。若公司网络限制了外部 API 访问（ip-api.com），位置功能会超时（5 秒）。不影响其他功能和上报，只是报告中的城市字段为空。另外，使用 VPN 时显示的是 VPN 出口城市，非真实位置。

❓ 企业微信没收到通知？

检查 Webhook 地址是否过期。可在企业微信群设置中重新生成 Webhook，然后更新代码中的 WEBHOOK_URL 常量。

❓ 智能表上报失败（840001）？

错误码 840001 表示 Webhook 无效或过期。需重新生成智能表 Webhook。注意：智能表失败不会影响企业微信通知，两者独立执行。

❓ 如何恢复误删的文件？

工具使用 Windows 回收站删除，可在桌面回收站中右键还原，无需担心永久丢失。

❓ 如何更新到新版本？

直接替换 .exe 文件即可，无需卸载旧版本。配置和历史记录存储在程序同目录。
